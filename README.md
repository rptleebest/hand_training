# 손 기능 재활 가상훈련 앱

MediaPipe Hands Web 기반의 뇌졸중 환자 손 기능 재활용 교육/피드백 앱입니다. Python MediaPipe, OpenCV, streamlit-webrtc를 사용하지 않고, 스마트폰/태블릿/PC 브라우저 안에서 손 landmark를 추정합니다.

## v8 주요 변경

- 물컵을 손에 잡아서 마시기 과제는 물컵을 손에 잡은 후 입쪽으로 물컵을 이동하면 성공하는 방식입니다。
- 공기방울 과제는 손 주변에 여러 개의 투명 공기방울이 나타나며, 환자가 손을 공기방울 위치로 이동한 뒤 손을 쥐어 터뜨리는 방식입니다.
- 공기방울 크기, 정지/움직임, 속도는 기존 공통 설정인 `물방울/공기방울/물컵 크기 보정`, `물방울/공기방울/물컵 속도 보정`, `난이도`에 의해 조절됩니다.
- 단계별 음성 안내를 강화했습니다.

## 포함 과제

1. 물방울 잡아 터뜨리기: 손 쥐기
2. 작은 물방울 집어 터뜨리기: 엄지-검지 집기
3. 손 펴기 훈련: 물방울 위에서 손 펴기
4. 가상 물컵 잡고 마시기
5. 손 주변 공기방울 잡아 터뜨리기

## 공기방울 과제 안내 흐름

- 손 주변의 공기방울 중 하나로 손을 천천히 가져가세요.
- 공기방울 안에서 손을 쥐세요.
- 잡은 상태를 잠시 유지하세요.
- 정상적으로 성공하면 다음 회차를 같은 방법으로 반복합니다.
- 시간이 너무 오래 걸리거나 손이 화면에서 사라지면 성공으로 기록하지 않고 같은 회차를 다시 수행하도록 안내합니다.

## 실행

```bash
python -m pip install -r requirements.txt
streamlit run app.py
```

## 배포

Streamlit Community Cloud에 배포할 때는 아래 파일을 저장소에 올립니다.

```text
app.py
requirements.txt
packages.txt
README.md
.streamlit/config.toml
assets/audio/README.md
scripts/generate_korean_audio_edge_tts.py
```

`requirements.txt`는 Streamlit만 필요합니다.

```text
streamlit>=1.38
```

`packages.txt`는 빈 파일입니다.

## 주의

이 앱은 교육용/피드백용 손 기능 재활 훈련 도구입니다. 단일 웹캠 기반 손 landmark 추정은 조명, 손 가림, 카메라 각도, 손 떨림, 마비 손의 변형에 영향을 받습니다. 임상 진단 또는 치료 효과 판정에 사용하려면 별도의 신뢰도·타당도 검증이 필요합니다.

## 주요 사항

- 카메라 버튼을 `① 카메라만 켜기`로 분리했습니다. 이 버튼은 카메라/MediaPipe 모델만 시작하며 물방울을 생성하지 않습니다.
- 물방울은 `⑤ 훈련 시작`을 누른 뒤에만 생성됩니다.
- `③ 손 편 상태 보정`, `④ 쥔/집은 상태 보정`은 훈련과 분리되어 보정 완료 후 다시 카메라 준비 화면으로 돌아갑니다.
- 두 보정이 모두 완료되지 않으면 훈련 시작을 막고 보정을 안내합니다.
- `훈련 중단`은 카메라는 유지하고 과제만 중단합니다. 옵션 변경 후 다시 훈련 시작을 누를 수 있습니다.
- 카메라만 켜진 상태에서는 혹시 이전 목표가 남아 있어도 매 프레임 target을 비워 물방울이 표시되지 않도록 방어 코드를 추가했습니다.

## assets/audio 폴더

삭제하지 않는 것을 권장합니다. 실제 음성 mp3/wav가 없어도 앱은 동작합니다. 이 폴더는 향후 스마트폰/태블릿에서 브라우저 음성합성이 불안정할 때 녹음 음성을 넣기 위한 선택 폴더입니다. GitHub는 빈 폴더를 저장하지 않으므로 README.md를 남겨 둡니다.


## v17 변경 사항
- 훈련 중단 시 물방울뿐 아니라 진행 중인 음성 안내와 대기 중인 음성 큐도 즉시 중단합니다.
- 실행 순서를 `카메라만 켜기 → 손 편 상태 보정 → 쥔 상태 보정 → 훈련 시작`으로 다시 정리했습니다.
- 손 편 상태와 쥔 상태 보정이 끝나면 다음 버튼을 누르도록 음성으로 안내합니다.
- 물방울 안에서 먼저 손 편 상태가 확인된 뒤, 같은 물방울 안에서 손쥐기 기준을 충족해야 물방울이 터지도록 수정했습니다.
- 보정 기준 기반 손 편 점수와 손쥐기 점수를 단계 안내에 표시합니다.


## v17 변경 사항
- 훈련 시작 버튼을 바로 눌러도 자동 손동작 기준으로 훈련할 수 있도록 수정했습니다.
- 기존 손 편/쥔 상태 보정 2개 버튼을 하나의 `③ 손가락 동작 보정` 버튼으로 통합했습니다.
- 보정은 손 펴기 3초 평균 → 손 쥐기 3초 평균 순서로 자동 진행됩니다.
- 앱 사용 방법을 소리 활성화/카메라 시작 시 쉽게 안내하도록 수정했습니다.
- 손동작 인정 기준은 낮을수록 작은 손가락 굽힘도 쉽게 인정하고, 높을수록 더 분명한 쥐기 동작이 필요합니다.
- 음성 중단, 오디오 파일 중단, 음성 큐 초기화 처리를 강화했습니다.
