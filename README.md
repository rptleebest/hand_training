# 손 기능 재활 가상훈련 Streamlit 앱 v8

MediaPipe Hands Web 기반의 뇌졸중 환자 손 기능 재활용 교육/피드백 앱입니다. Python MediaPipe, OpenCV, streamlit-webrtc를 사용하지 않고, 스마트폰/태블릿/PC 브라우저 안에서 손 landmark를 추정합니다.

## v8 주요 변경

- 기존 `가상 물컵에 물 따르고 잡고 마시기` 과제를 제거하고 `손 주변 공기방울 잡아 터뜨리기` 과제로 변경했습니다.
- 공기방울 과제는 손 주변에 여러 개의 투명 공기방울이 나타나며, 환자가 손을 공기방울 위치로 이동한 뒤 손을 쥐어 터뜨리는 방식입니다.
- 공기방울 크기, 정지/움직임, 속도는 기존 공통 설정인 `물방울/공기방울/물컵 크기 보정`, `물방울/공기방울/물컵 속도 보정`, `난이도`에 의해 조절됩니다.
- 단계별 음성 안내를 강화했습니다.

## 포함 과제

1. 물방울 잡아 터뜨리기: 손 쥐기
2. 작은 물방울 집어 터뜨리기: 엄지-검지 집기
3. 손 펴기 훈련: 물방울 위에서 손 펴기
4. 가상 물컵 잡고 마시기
5. 손 주변 공기방울 잡아 터뜨리기

## 공기방울 과제 안내 흐름

- 손 주변의 공기방울 중 하나로 손을 천천히 가져가세요.
- 공기방울 안에서 손을 쥐세요.
- 잡은 상태를 잠시 유지하세요.
- 정상적으로 성공하면 다음 회차를 같은 방법으로 반복합니다.
- 시간이 너무 오래 걸리거나 손이 화면에서 사라지면 성공으로 기록하지 않고 같은 회차를 다시 수행하도록 안내합니다.

## 실행

```bash
python -m pip install -r requirements.txt
streamlit run app.py
```

## 배포

Streamlit Community Cloud에 배포할 때는 아래 파일을 저장소에 올립니다.

```text
app.py
requirements.txt
packages.txt
README.md
.streamlit/config.toml
assets/audio/README.md
scripts/generate_korean_audio_edge_tts.py
```

`requirements.txt`는 Streamlit만 필요합니다.

```text
streamlit>=1.38
```

`packages.txt`는 빈 파일입니다.

## 주의

이 앱은 교육용/피드백용 손 기능 재활 훈련 도구입니다. 단일 웹캠 기반 손 landmark 추정은 조명, 손 가림, 카메라 각도, 손 떨림, 마비 손의 변형에 영향을 받습니다. 임상 진단 또는 치료 효과 판정에 사용하려면 별도의 신뢰도·타당도 검증이 필요합니다.


## v9 변경 사항
- 실제 둥근 물방울 스타일 개선
- 물방울을 얼굴 아래쪽에 여러 방향으로 배치
- 설정한 총 개수만큼 한 번에 생성하고, 하나씩 터뜨리면 사라짐
- 모든 물방울이 사라지면 자동 종료
- 성공 횟수, 실패 횟수, 평균 성공시간을 화면과 TXT에 출력
- 같은 물방울 위에 머물러도 성공 횟수가 반복 증가하지 않도록 수정
