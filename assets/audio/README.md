# 선택 사항: 녹음 음성 파일

이 폴더는 삭제하지 않는 것을 권장합니다. GitHub는 빈 폴더를 보존하지 않으므로 README.md를 남겨둡니다.

스마트폰/태블릿에서 브라우저 음성합성이 불안정할 때, 아래 이름으로 mp3/wav 파일을 넣으면 녹음 음성이 우선 재생됩니다.

예: sound_test.mp3, start_camera.mp3, open_cal_start.mp3, close_cal_start.mp3, training_start.mp3, training_abort.mp3, complete.mp3

파일이 없어도 앱은 브라우저 음성합성과 신호음/진동/화면 안내로 동작합니다.
